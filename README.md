# Enerdy Demand vs Economic Indicators Pipeline

This project is to analyze how energy consumption trends change alongside economic indicators such as ub employment rates using
public US/EU datasets.

## Project Goal
The goal is to build a simple data pipeline that:
-Extracts public energy and economic data
-Cleans and aligns time-series data
-Stores the Results in a SQL database
-Produce basic analytical insigts

##Tech Stack
-Python
-SQL (MySql)
-Airflow (for scheduling)
